2. Central Tendency: Sit-up Scores

This example shows how to calculate Mean, Median, and Mode for sit-up performance data.

import numpy as np
from scipy import stats

# Sit-up data
situps = np.array([2, 2, 6, 6, 12, 12, 14, 15, 15, 15, 24, 25])

# Mean
mean = np.mean(situps)

# Median
median = np.median(situps)

# Mode
mode = stats.mode(situps, keepdims=True)

# Display results
print(f"Mean: {mean}")
print(f"Median: {median}")
print(f"Mode: {mode.mode[0]} (Frequency: {mode.count[0]})")

3. Measures of Dispersion: Company A vs Company B

This example calculates Range, MAD, Standard Deviation, Variance, and IQR for two companies' profit data.

import numpy as np
import pandas as pd

# Profit data in millions
company_a = np.array([43, 44, 0, 25, 20, 35, -8, 13, -10, -8, 32, 11, -8, 21])
company_b = np.array([17, 15, 12, 17, 15, 18, 12, 15, 12, 13, 18, 18, 14, 14])

# Create DataFrame
df = pd.DataFrame({
    'Company A': company_a,
    'Company B': company_b
})

# Mean
mean_a = np.mean(company_a)
mean_b = np.mean(company_b)

# Range
range_a = np.ptp(company_a)
range_b = np.ptp(company_b)

# Mean Absolute Deviation
mad_a = np.mean(np.abs(company_a - mean_a))
mad_b = np.mean(np.abs(company_b - mean_b))

# Standard Deviation
std_a = np.std(company_a)
std_b = np.std(company_b)

# Variance
var_a = np.var(company_a)
var_b = np.var(company_b)

# Interquartile Range
q1_a = np.percentile(company_a, 25)
q3_a = np.percentile(company_a, 75)
iqr_a = q3_a - q1_a

q1_b = np.percentile(company_b, 25)
q3_b = np.percentile(company_b, 75)
iqr_b = q3_b - q1_b

# Display Results
results = pd.DataFrame({
    'Measure': ['Mean', 'Range', 'MAD', 'Standard Deviation', 'Variance', 'IQR'],
    'Company A': [mean_a, range_a, mad_a, std_a, var_a, iqr_a],
    'Company B': [mean_b, range_b, mad_b, std_b, var_b, iqr_b]
})

print(results)
